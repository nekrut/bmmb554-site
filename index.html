How does this class work?

Our course is divided into sixteen topics. Each topic is roughly equivalent to two 2-hour lectures. Every Wednesday we will be posting new slides () and videos () corresponding to a single section (you will also receive an e-mail notification). In addition, we will provide links to a set of key publications relevant for a given topic. You are expected to read these. Homeworks will be published on-line either the same Wednesday or the following Thursday. You will have one week to complete and submit the homework for the evaluation.
